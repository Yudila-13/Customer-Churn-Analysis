# Customer-Churn-Analysis.

# Project Overview.
This project analyzes customer churn behavior using exploratory data analysis (EDA) and machine learning. The goal is to identify key factors influencing customer churn and provide business insights.

# Dataset.
- Records: 7043
- Features: 22
- Target Variable: StateNextMonth (Churn / No Churn)

# Key Steps.
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Encoding
- Logistic Regression Model
- Feature Importance Analysis

# Key Insights.
- Fiber optic internet users have higher churn rates
- Electronic check payment method strongly correlates with churn
- Customers with higher monthly charges tend to churn more

# Tools and Technologies.
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

# Model Used.
- Logistic Regression

# Results.
The model identifies important churn drivers and provides actionable business insights.
